MERCADOHOY

Aplicación web ecommerce para comparación de precios entre supermercados.

Descripción

MERCADOHOY es un sistema web desarrollado con tecnologías MERN que permite a los usuarios comparar precios de productos, organizar listas de compras y gestionar un carrito de compras.

Tecnologías utilizadas
React
Vite
TailwindCSS
Node.js
Express
MongoDB
React Router
Context API
Características principales
Comparador de precios.
Carrito de compras.
Sistema de favoritos.
Gestión de listas.
Login y registro.
Diseño responsive.
Interfaz moderna ecommerce.
